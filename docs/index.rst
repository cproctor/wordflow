Wordflow
========

`Wordflow` provides several simplified command-line utilities designed to be
composed with UNIX pipes. Wordflow is distributed via PyPI, so you can install
it using pip or poetry.

This documentation is organized according to a common pattern: the
:ref:`introduction` contains narrative documentation walking you through
how to use the utilities. The :ref:`api` contains a complete description of
each command and how to use it. If you're trying to learn what `Wordflow`
can do, start with the :ref:`introduction`. If you're trying to check a
detail about how to use a particular command, head to the :ref:`api`.

`Wordflow` is part of the `Making With Code` introductory computer science
curriculum, with an intended audience of beginners. If you are interested
in `Making With Code`, or if you found `Wordflow` helpful,
`Dr. Chris Proctor <https://chrisproctor.net>`_ would love to hear from you.
